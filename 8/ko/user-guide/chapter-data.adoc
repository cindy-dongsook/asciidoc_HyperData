== 데이터 

본 장에서는 데이터 카테고리 하위 각 메뉴의 사용법에 대해서 설명한다.

////
 ---------------------

 ---------------------

=== 개요

HyperData Web UI의 데이터 카테고리에서 HyperData는 가상화를 통해 분산된 데이터를 통합하고 공유할 수 있다.

다음은 데이터 카테고리의 하위 메뉴에 대한 설명이다.

[width="100%",cols="3,7",options="header"]
|============================================
|메뉴|설명
|<<sect_data_datasource,Data Source>> |자체적인 디폴트 데이터소스 외에도 외부 소스를 등록하여 관리한다.
|<<sect_data_meta_mgmt,Meta Management>> |프로젝트 내 등록된 데이터 객체와 컬럼에 대한 메타 데이터를 관리한다.
|<<sect_data_shared_data,Shared Data>> |공유된 데이터 객체 현황을 조회한다.
|<<sect_data_data_asset,Data Asset>> |프로젝트 내 등록된 데이터 객체에 대한 검색을 수행한다.
|============================================
