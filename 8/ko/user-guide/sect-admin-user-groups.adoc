[#sect_admin_user_groups]
=== User Groups
 
HyperData에서는 2인 이상의 사용자를 묶어 하나의 사용자 그룹을 생성할 수 있다. 이를 통해 프로젝트 구성원을 수정할 때 그룹 단위로 추가/삭제가 가능하다. +
[시스템 관리] > [User Groups] 메뉴를 선택하면 사용자 그룹을 조회하고 관리할 수 있다. +
사용자 그룹 화면에서 [+그룹] 버튼을 클릭하면 신규 사용자 그룹을 생성할 수 있다. 사용자 그룹 목록의 '실행' 항목에서 image:resources/icon_edit.png[width=20] 아이콘을 클릭하면 사용자 그룹을 수정할 수 있다. 사용자 그룹 목록에서 그룹을 선택한 후 [삭제] 버튼을 클릭하거나 '실행' 항목에 image:resources/icon_delete.png[width=20] 아이콘을 클릭하면 사용자 그룹을 삭제할 수 있다.

목록 위에 '검색' 입력항목에 검색할 그룹 이름을 입력하고 <Enter> 키를 누르거나 image:resources/icon_search.png[width=22] 아이콘을 클릭하면 해당 그룹 정보를 조회할 수 있다. '이름' 항목을 클릭하면 <<figure_user_group_detail,User Group Detail 화면>>에서 해당 그룹의 기본 정보를 조회할 수 있다. 목록 아래에 조회된 건수와 페이징 정보, 조회할 건수를 선택할 수 있다.

[#figure_user_group_management]

==== 사용자 그룹 생성

<<figure_user_group_management,사용자 그룹 화면>>에서 [+그룹] 버튼을 클릭하면 사용자 그룹을 신규로 생성할 수 있다.

New User Group 화면에서 그룹 정보를 설정한다(* 표시 항목은 필수 입력). '이름' 항목은 중복된 값을 입력하면 에러가 발생하므로 주의한다. [+참여자] 버튼을 클릭한 후 사용자 목록을 조회해서 해당 그룹에 사용자를 추가할 수 있다. 이미 추가된 사용자는 목록에서 선택한 후 [삭제] 버튼을 클릭하면 그룹에서 삭제할 수 있다. 각 항목을 입력한 후 [추가] 버튼을 클릭한다.

image::resources/figure_user_group_add.png[title="사용자 그룹 생성", width=500]

==== 사용자 그룹 수정

<<figure_user_group_management,사용자 그룹 화면>>에서 '실행' 항목의 image:resources/icon_edit.png[width=20] 아이콘을 클릭하면 그룹 정보를 수정할 수 있다.

Eidt User Group 화면에서 [+참여자] 버튼을 클릭한 후 사용자 목록을 조회해서 해당 그룹에 추가할 사용자를 선택한다. 이미 추가된 사용자는 목록에서 선택한 후 [삭제] 버튼을 클릭하면 그룹에서 삭제할 수 있다. 항목을 수정한 후 [저장] 버튼을 클릭한다.

[#figure_user_group_edit]
image::resources/figure_user_group_edit.png[title="사용자 그룹 수정", width=500]

==== 사용자 그룹 삭제

<<figure_user_group_management,사용자 그룹 화면>>의 그룹 목록을 선택한 후 [삭제] 버튼을 클릭하거나 '실행' 항목의 image:resources/icon_delete.png[width=20] 아이콘을 클릭하면 그룹 정보를 삭제할 수 있다. 목록에서 삭제할 정보를 확인한 후 [삭제] 버튼을 클릭한다.
 
image::resources/figure_user_group_delete.png[title="사용자 그룹 삭제", width=300]

==== 사용자 그룹 상세 조회

<<figure_user_group_management,사용자 그룹 화면>> 목록의 '이름' 항목을 클릭하면 User Group Detail 화면에서 해당 그룹의 기본 정보를 조회할 수 있다.

[수정] 버튼을 클릭하면 <<figure_user_group_edit, Edit User Group 화면>>으로 이동해서 사용자 그룹 정보를 수정할 수 있다.
