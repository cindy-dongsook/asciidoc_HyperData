[#sect_admin_users]
=== Users

[시스템 관리] > [Users] 메뉴를 선택하면 사용자 화면으로 이동해서 현재 HyperData 사용자를 조회하고 관리할 수 있다.

[+사용자] 버튼을 클릭하면 신규 사용자를 생성할 수 있다. 사용자 목록의 '실행' 항목에서 image:resources/icon_edit.png[width=20] 아이콘을 클릭하면 사용자 정보를 수정할 수 있다. 사용자 목록에서 사용자를 선택한 후 [삭제] 버튼을 클릭하거나 '실행' 항목에 image:resources/icon_delete.png[width=20] 아이콘을 클릭하면 사용자를 삭제할 수 있다.

목록 위에 '검색' 입력항목에 검색할 사용자 ID를 입력하고 <Enter> 키를 누르거나 image:resources/icon_search.png[width=22] 아이콘을 클릭하면 해당 ID의 사용자 정보를 조회할 수 있다. 'ID' 항목을 클릭하면 <<figure_user_detail,User Details 화면>>에서 해당 사용자 기본 정보를 조회할 수 있다. 목록 아래에 조회된 건수와 페이징 정보, 조회할 건수를 선택할 수 있다.

[#sect_user_create]
==== 사용자 생성

<<figure_users_management,사용자 화면>>에서 [+사용자] 버튼을 클릭하면 사용자를 신규로 생성할 수 있다. 각 항목 정보를 입력한 후 [추가] 버튼을 클릭한다.


image::resources/figure_user_create.png[title="사용자 생성", width=350]

다음은 각 항목에 대한 설명이다. (* 표시 항목은 필수 입력)

[width="100%",cols="3,8a",options="header"]
|===============
|항목		        |설명
|ID *				|사용자 ID를 입력한다. +
ID는 영문자(대소문자 구분)과 숫자의 조합으로 생성할 수 있으며, 특수문자
(_, -)의 사용이 가능하다. +
*중복된 사용자 ID를 입력하면 에러가 발생하므로 주의한다.*
|비밀번호 *		    |사용자 비밀번호를 입력한다. +
비밀번호는 국내 비밀번호 가이드 라인에 따라 8자 이상의 영문, 숫자, 특수
문자( !, @, #, $, %, ^, *, +, =, -) 조합으로 설정해야 한다.
|비밀번호 확인 *	|사용자 비밀번호를 다시 한번 입력한다.
|이름 *				|사용자 이름을 입력한다.
|이메일				|사용자 이메일을 입력한다
|전화번호			|사용자 연락처를 입력한다.
|역할				|사용자의 권한 정보를 입력한다. +
– administrator : 관리자 +
– user : 일반 사용자 +
관리자를 제외한 사용자에게는 'user'만 선택 가능하다.
|사용자 그룹		|사용자가 속할 그룹을 선택한다(그룹 선택 기능은 추후 제공 예정).
|프로젝트			|사용자가 속할 프로젝트를 선택한다(프로젝트 선택 기능은 추후 제공 예정).
|===============

[#sect_user_edit]
==== 사용자 수정

<<figure_users_management,사용자 화면>>의 사용자 목록에서 '실행' 항목의 image:resources/icon_edit.png[width=20] 아이콘을 클릭하면 사용자 정보를 수정할 수 있다. 입력 가능한 항목을 수정한 후 [저장] 버튼을 클릭한다. 각 항목에 대한 자세한 내용은 <<sect_user_create,"2.2.1. 사용자 생성">>을 참고한다.

메인 화면의 <<figure_userinfo, 그림 1.8. 사용자 정보 조회>>에서 [사용자 설정] 버튼을 클릭해도 로그인 중인 계정을 수정할 수 있다.

[#figure_user_edit]
image::resources/figure_user_edit.png[title="[Users] - 사용자 수정", width=350]

==== 사용자 삭제
<<figure_users_management,사용자 화면>>의 사용자 목록에서 사용자를 선택한 후 [삭제] 버튼을 클릭하거나 '실행' 항목의 image:resources/icon_delete.png[width=20] 아이콘을 클릭하면 사용자를 삭제할 수 있다. 목록에서 삭제할 정보를 확인한 후 [삭제] 버튼을 클릭한다. 관리자인 admin 계정은 삭제할 수 없다.

image::resources/figure_user_delete.png[title="사용자 삭제", width=300]

[#sect_user_detail]
==== 사용자 정보 상세조회

<<figure_users_management,사용자 화면>>의 목록에 'ID' 항목을 클릭하면 User Details 화면에서 해당 사용자 기본 정보를 조회할 수 있다. 각 항목에 대한 자세한 내용은 <<sect_user_create,"2.2.1. 사용자 생성">>을 참고한다.

[수정] 버튼을 클릭하면 <<figure_user_edit, Edit User 화면>>으로 이동해서 사용자 정보를 수정할 수 있다. [삭제] 버튼을 클릭하면 조회한 사용자 정보를 삭제할 수 있다.

[#figure_user_detail]
image::resources/figure_user_detail.png[title="사용자 정보 상세조회", width=350]
