[#sect_analytics_ml_auto]
=== AutoML 

분석 > [AutoML] 메뉴를 선택하면 데이터와 분석 목표만을 입력해서 자동으로 모델을 생성할 수 있다. 자동 모델 생성은 데이터 준비, 실험 구성, 목적 선택의 과정으로 진행된다.(((AutoML)))

==== 데이터 준비


데이터 준비단계에서는 HyperData의 데이터 오브젝트 목록을 조회한다. 입력할 데이터 오브젝트를 선택하고 [다음] 버튼을 클릭한다. 실험 생성을 중단하려면 [취소] 버튼을 클릭한다. 실험 중단되면 <<figure_automl_experiment, Experiments 목록 화면>>으로 이동한다.

====  실험 구성

실험 구성단계에서는 데이터 준비단계에서 선택한 데이터 오브젝트의 데이터를 조회한다. 실험의 이름과 대상열을 선택한 후 [다음] 버튼을 클릭한다. 이전 단계로 이동하려면 [이전] 버튼을 클릭하고, 실험 생성을 중단하려면 [취소] 버튼을 클릭한다. 실험 중단되면 <<figure_automl_experiment, Experiments 목록 화면>>으로 이동한다.
 
image::resources/figure_automlcol_selection.png[title="실험 구성 단계", width=750]

다음은 각 항목에 대한 설명이다.
[width="100%",cols="3,7",options="header"]
|===========
|항목	     |설명
|데이터 필드 | 선택한 데이터 오브젝트의 필드 정보와 저장된 데이터를 조회한다.
|데이터      |이전 단계에서 선택한 데이터 오브젝트의 이름이 출력된다.
|실험이름    |진행할 실험 이름을 입력한다. 입력한 이름은 이전의 저장한 실험이름과 중복될 수 없다.
|대상 열 | 실험 목적에 맞는 분석 대상 컬럼을 선택한다.
|===========

데이터 준비단계에서 시계열 데이터 오브젝트를 선택한 경우 시계열 분석을 위한 추가적인 설정이 필요하다.

image::resources/figure_automl_timeseries1.png[title="시계열 데이터의 실험 구성 단계", width=750]

image::resources/figure_automl_timeseries2.png[title="시계열 데이터의 추가 설정", width=750]

다음은 각 항목에 대한 설명이다.

[width="100%",cols="3,7",options="header"]
|===========
|항목		|설명
|시계열 컬럼|시간 컬럼을 선택한다.
|그룹		|범주형 컬럼을 그룹으로 묶어 처리할 수 있다. 싱글 그룹인 경우 설정할 필요없다.
|===========

====  목적 선택

설정한 실험의 모델의 유형을 선택한다. 연속적이지 않은 범주형 열을 예측하는 경우 분류 모델(Classification Model)을 선택하고, 연속적인 수치형 열을 예측하는 경우는 회귀 모델(Regression Model)을 선택한다.

이전 단계로 이동하려면 [이전] 버튼을 클릭하고, 실험 생성을 중단하려면 [취소] 버튼을 클릭한다. 실험 중단되면 <<figure_automl_experiment, Experiments 목록 화면>>으로 이동한다.

모델의 유형을 선택한 후 [실험시작] 버튼을 클릭하면 학습이 시작된다.

image::resources/figure_automltask_selection.png[title="목적 선택 단계", width=750]

데이터의 크기에 따라 하이퍼파라미터 최적화 과정에서 긴 시간이 소요될 수 있으며 동시에 여러 실험을 생성한 경우 자원 부족으로 크게 느려질 수 있다. 학습시간 경고 알람이 뜨면 [계속] 버튼을 클릭한다(버튼을 클릭하지 않아도 실험은 내부적으로 수행된다). 실험이 완료되면 <<figure_automl_experiment, Experiments 목록 화면>>으로 이동한다.

image::resources/figure_automlcol_warning.png[title="학습 시간 경고창", width=300]
