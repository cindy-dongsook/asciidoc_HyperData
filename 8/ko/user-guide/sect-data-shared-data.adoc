[#sect_data_shared_data]
===  Shared Data  

[데이터] > [Shared Data] 메뉴를 선택하면 프로젝트 내에 공유되는 데이터 객체 정보를 조회할 수 있다. 데이터 객체의 공유 설정에 대한 자세한 내용은 <<sect_dataobject_sharing,"데이터 객체 공유">를 참고한다. 목록 아래에 조회된 건수와 페이징 정보, 조회할 건수를 선택할 수 있다.

다음은 프로젝트 SEARCH_TEST에서 생성된 데이터 객체 고객정보를 프로젝트 JY_TEST에 공유한 내역이다. 다른 프로젝트에서 SEARCH_TEST 프로젝트로 공유된 데이터 객체가 있다면, 다른 프로젝트로부터 공유에 해당 정보가 조회된다.
