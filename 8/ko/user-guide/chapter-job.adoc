=== 개요

JOB 카테고리에서는 HyperData Web UI의 Data Job, Service Job을 관리하고 Data Flow, Service Flow를 만들고 분석한다.

다음은 JOB 카테고리의 하위 메뉴에 대한 설명이다.

[width="100%",cols="3,8",options="header"]
|================================================
|메뉴	| 설명
|<<sect_job_data_job,Data Job>>				|등록된 데이터소스로부터 데이터 가공 및 분석을 작업을 수행한다. 작성된 Data Job은 Service Job에서 사용된다. 
|<<sect_job_service_job,Service Job>>	|Service Job을 생성하고 작성한 후 그룹 분석/가공 작업을 정의하고 작업을 수행한다. Service Job을 정의한 후 Scheduler 등에서 활용 가능하다.
|<<sect_job_job_mornitoring,Job Monitoring>>|Data Job, Service Job의 수행 히스토리를 확인한다.
|================================================
