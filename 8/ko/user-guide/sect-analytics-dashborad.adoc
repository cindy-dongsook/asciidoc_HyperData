[#sect_analytics_dashboard]
=== Visualization 

분석 > [Visualization] 메뉴를 선택하면 생성된 대시보드의 목록을 조회할 수 있다. 각 대시보드의 용도에 따라서 시각화 결과물들을 생성 및 분석하기 위한 화면을 구성할 수 있다.

대시보드 화면의 목록에서 '이름' 항목을 선택하면 대시보드를 구성할 수 있는 <<figure_card_setting,대시보드 레이아웃 편집 화면 >>으로 이동한다. 자세한 내용은 <<sect_dashbord_card_setting,"대시보드 레이아웃 편집">>을 참고한다.

목록 위에 '검색' 입력항목에 대시보드 이름을 입력하고 <Enter> 키를 누르거나 image:resources/icon_search.png[title="", width=22] 아이콘을 클릭하면 해당 이름의 대시보드를 검색할 수 있다. 목록 아래에 조회된 건수와 페이징 정보, 조회할 건수를 선택할 수 있다.

==== 대시보드 등록


<<figure_dashboard,대시보드 화면 >>에서 [+생성] 버튼을 클릭하면 대시보드를 생성할 수 있다.

대시보드 등록 화면에서 대시보드의 정보를 입력한 후 [등록] 버튼을 클릭한다. 대시보드를 생성한 후 해당 조회할 차트와 테이블을 추가해야 한다. 자세한 내용은 <<sect_dashbord_card_setting, "대시보드 레이아웃 편집">>을 참고한다.

==== 대시보드 이미지 관리

<<figure_dashboard, 대시보드 화면>>의 목록에 '실행' 항목의 image:resources/icon_screenshot.png[width=20] 아이콘을 클릭하면 해당 대시보드 내 카드 이미지를 관리할 수 있는 대시보드 이미지 관리 화면으로 이동한다.

[#sect_dashbord_card_setting]
==== 대시보드 레이아웃 편집


대시보드를 생성한 후 조회할 시각화 개체인 차트와 테이블을 추가하거나 속성을 수정해서 화면의 레이아웃을 구성할 수 있다.사용자의 용도에 따라 다양한 대시보드를 생성하여 데이터 분석을 수행할 수 있다.

<<figure_dashboard, 대시보드 화면>>의 목록에서 '이름' 항목을 선택하면 대시보드 레이아웃 편집 초기화면으로 이동한다.

image::resources/figure_card_emptydashboard.png[title="대시보드 레이아웃 편집 초기 화면", width=750]

대시보드 상단 버튼을 클릭해서 대시보드를 수정, 저장, 공유하는 기능을 설정할 수 있다.

[width="100%",cols="3,7",options="header"]
|================
|버튼       |설명
|[수정]     |대시보드에서 조회할 차트, 테이블 등을 추가 및 삭제할 수 있다.
|[이미지 저장]|대시보드를 이미지파일(확장자 .png)로 저장할 수 있다.
|[엑셀 파일 저장]|대시보드를 엑셀파일(확장자 .xlsx)로 저장할 수 있다. 각각의 차트, 데이블은
이미지로 저장된다. 이때 트리 테이블, 원본 테이블의 경우 해당 테이블의 데
[엑셀 파일 저장]
이터는 새로운 시트로 추가되어 저장된다(파일 저장은 hd_config.properties
에 지정한 값에 따라 해당 개수만 다운로드가 가능하다).
|================

대시보드 레이아웃 편집 화면에서 대시보드에 추가할 차트와 테이블을 추가하고 부가적인 속성을 설정한다. 레이아웃 구성이 완료되면 [저장] 버튼을 클릭한다.

image::resources/figure_card_setting.png[title="대시보드 레이아웃 편집 화면", width=750]

[iframe Export] 스위치를 클릭하면 대시보드 레이아웃 편집 화면에서 대시보드를 외부로 공유하는 Iframe 기능을 활성화할 수 있다. 기능을 활성화하면 Iframe 태그로 이루어진 HTML 코드를 제공한다.

Iframe HTML 코드는 대시보드에 설정된 크기 및 스타일에 맞춰 생성된다.

image::resources/figure_card_iframe_export.png[title="대시보드 iframe export 설정 화면", width=750]

[Copy] 버튼을 클릭해서 해당 코드를 복사한 후 표시할 웹 페이지의 HTML 코드 내에 추가하면 브라우저 내에 HyperData 대시보드를 삽입할 수 있다.

image::resources/figure_card_iframe_connection.png[title="대시보드 iframe 접속화면", width=750]

* ① 레이아웃 속성 영역
+
대시보드의 속성과 선택한 카드의 속성을 설정하는 영역이다. 대시보드의 가로, 세로 크기와 배경색을 설정할 수 있고 대시보드에 카드를 추가한 경우 선택한 카드의 속성을 설정할 수 있다. 카드 속성에 대한 설명은 각 절의 설명을 참고한다.
+
[width="100%",cols="3,7",options="header"]
|=============
|유형       |설명
|대시보드 크기|대시보드의 가로, 세로 크기를 조절할 수 있다. 가로, 세로 각각 그리드 개수 30
개가 최소값이며 해당 값 이하인 경우 최소값으로 크기가 설정된다.
|배경 색상  |대시보드의 배경 색상을 설정할 수 있다. 팔레트 버튼을 클릭하면 Color Picker
다이얼로그가 뜨며, 해당 다이얼로그에서 원하는 색상을 선택할 수 있다.
|배경 이미지|대시보드에 배경 이미지를 삽입하는 기능이다. [Choose a file] 버튼을 클릭 후
나타나는 다이얼로그로 원하는 이미지를 선택할 수 있다. 이미지 선택후에 나타
나는 반복/맞춤 설정 라디오버튼을 통해 이미지를 반복하거나 대시보드 크기에
맞추도록 설정할 수 있다.
|=============

|글로벌 필터|여러 차트 및 테이블에 동시에 필터를 적용, 조절하는 경우 선택한다. 자세한 내용은 “6.2.5.3. 대시보드 카드 - 글로벌 필터”를 참고한다.

[#sect_dashbordcard_chart]
===== 대시보드 카드 - 차트

데이터를 막대, 꺾은선, 원형, 계층, 분포 등 다양한 차트로 시각화하는 차트 카드를 생성할 수 있다.

<<figure_card_composition,대시보드 레이아웃 편집 화면>>에서 [카드 추가] 버튼을 클릭한 후 드롭다운 목록에서 생성할 차트 종류를 선택한다. 차트 종류를 선택하면 빈 차트 카드가 대시보드의 작업 영역에 생성된다. 생성된 차트를 선택하면 왼쪽에 레이아웃 설정 영역의 [차트 데이터], [차트 스타일] 탭에서 상세 정보를 설정할 수 있다.

image::resources/figure_card_emptycard_1.png[title="대시보드 레이아웃 편집 - 차트 카드 편집", width=750]

====== [차트 데이터] 탭 
 
레이아웃 설정 영역의 [차트 데이터] 탭을 선택하면 차트로 시각화할 대상 데이터 객체, 데이터 필드를 선택하여 차트를 구성할 수 있다.
 
image::resources/figure_card_chartdata_1.png[title="대시보드 레이아웃 편집 - 차트 데이터 (1)", width=330]
 
'데이터 객체' 항목에서 [선택] 버튼을 클릭해서 차트를 구성할 데이터 객체를 선택하면 '데이터 객체' 항목에 데이터 오브젝트 이름이 나타나며, 분석 데이터의 '데이터 필드' 항목에 데이터 객체를 구성하는 필드의 이름과 자료형이 아이콘으로 표현된다.
 
image::resources/figure_card_chartdata_2.png[title="대시보드 레이아웃 편집 - 차트 데이터 (2)", width=330]
 
차트 구성 요소(카테고리, 범례, 측정 값, 데이터 필터)는 선택한 차트의 종류에 따라 달라지며, 각 요소에 여러 개의 필드를 추가할 수 있다. 차트 요소 항목에 추가할 필드를 선택한 후 드래그 앤 드롭하면 필드가 속성에 설정된다.
 
다음은 각 차트별 설정 항목에 대한 설명이다. 각 항목에 대한 상세 설정방법은 <<sect_dashbordcard_chart_item, "대시보드 카드 상세 설정">>을 참고한다.

* Measure Option*

차트 요소에 추가한 필드의 자료형(data type)이 숫자(numeric)인 경우 해당 필드를 Measure(측정 값), X Axis(X축 값), Y Axis(Y축 값)에 추가할 수 있다. 필드를 요소에 추가한 후 클릭하면 Measure Option에서 'Aggregation Function'을 선택할 수 있다. Aggregation Function은 카테고리 및 범례의 Group by function에 따른 범주 및 계열별 집계를 수행한다.

image::resources/figure_card_measureopt.png[title="대시보드 레이아웃 편집 - 차트 데이터 설정 - Measure Option", width=260]

** Aggregation Function
+
다음은 Aggregation Function에 대한 설명이다. 데이터 필드의 자료형(data type)이 숫자가 아닌 경우 COUNT로 고정된다.
+
[width="100%",cols="3,7",options="header"]
|=======
|항목   |설명
|SUM    |총 합
|MAX    |최대값
|MIN    |최소값
|COUNT  |데이터 개수
|AVERAGE|평균값
|=======
** Number Format
+
다음은 Number Format에 대한 설명이다. 데이터 필드의 자료형(data type)이 숫자인 경우 설정이 가능하다.
+
[width="100%",cols="3,7",options="header"]
|===============
|항목           |설명
|general 1234   |일반 숫자
|number 1,234   |콤마(,)가 있는 숫자
|percent 123400%|백분율로 표현된 숫자
|===============
* Filter Option 
+
필드를 데이터 필터 요소에 추가한 후 클릭하면 Filter Option에서 필터 이름과 기준을 설정할 수 있다.
+
image::resources/figure_card_compopt.png[title="대시보드 레이아웃 편집 - 차트 데이터 설정 - Component Option", width=250]
+
다음은 각 항목에 대한 설명이다.
+
[width="100%",cols="2,8a",options="header"]
|===================================================
|항목       |설명
|Alias      |필터 이름을 설정할 수 있다. 기본값은 선택한 '(데이터 필드 이름) filter'이다.
|Filter     |한 개 이상의 필터 기준을 설정한다.
|Option \{_필터 기준으로 설정된 데이터 필드의 이름_}|두 개 이상의 필터 기준을 설정하는 경우 'add OR filter option'을 통해 OR 연산으로 필터를 추가할 수 있다(AND 연산으로 추가할 경우 차트 데이터 사
이드바의 Filter 칸에 추가로 데이터 필드 블록을 위치시킨다). +
각 필터 설정 사항은 [∧] , [∨] 버튼을 클릭해서 숨기거나 보이게 할 수 있으
며, 두 개 이상의 필터를 설정하는 경우 해당 필터 이름의 오른쪽에 [X] 버튼
을 클릭해서 삭제할 수 있다. +
– Field : 드래그 앤 드롭으로 'Field' 항목에 필터 기준으로 삼을 데이터 필드
를 위치시킬 수 있다. +
– Function : 자료형(data type)에 따라 적절한 Function을 드롭다운 목록에
서 선택할 수 있다. 숫자에 대해 Equal, Not Equal, Range, 그 외 자료형에
대해 Equal, Not Equal 함수를 지원한다. +
• Equal : 체크박스로 선택된 값에 해당하는 데이터를 카드에 반영한다. +
• Not Equal : 체크박스로 선택된 값을 제외한 데이터를 카드에 반영한다. +
• Range : 기입한 범위 내 데이터를 카드에 반영한다.
|===================================================
* Component Option
+
차트 요소에 필드를 추가한 후 image:resources/icon_configuration.png[width=20] 버튼을 클릭하면 Component Option에서 데이터 정렬, 범례 색 등을 설정할 수 있다.
+
image::resources/figure_card_filteropt.png[title="대시보드 레이아웃 편집 - 차트 데이터 설정 - Filter Option", width=280]
+
다음은 각 항목에 대한 설명이다.
+
[width="100%",cols="2,8a",options="header"]
|============
|항목       	|설명
//-----------------------
|Order Option	|정렬 기준을 설정할 수 있다. +
– Order Type : ASC/DESC 설정에 따라 오름차순/내림차순으로 정렬된다. +
– Ordering By Column : 다른 컬럼에 대한 집계값으로 정렬할 수도 있다. + 
– Aggregation Function : 정렬기준으로 이용할 컬럼을 'Field' 항목에 드래그
앤 드롭으로 설정하면 Measure Option([그림 6.19])와 같이 데이터 타입
에 따라 집계함수를 설정할 수 있다.
//-----------------------
|Color Option	|'Coloring By Column' 항목을 On/Off 토글로 설정한다. +
– Off : 각 범례 오른쪽 팔레트를 클릭하면 생성되는 Color Picker 다이얼로그
를 통해 범례 개별 색상을 설정할 수 있다. +
– On : 지정된 필드의 집계값을 기준으로 색상을 설정할 수 있다. 기준색상
을 선택하면 집계 값을 기준으로 값이 작으면 옅은 색상으로, 값이 크면 짙
은 색상으로 표시된다. +
막대그래프에서는 On/Off 설정이 가능하며, 꺾은선 - 기본, 누적, 100%누적,
레이더 / 원형 - 기본 / 분포 - 기본, 버블 차트에서는 Off 기능만 제공한다.
