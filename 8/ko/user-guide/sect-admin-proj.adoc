[#sect_admin_proj]
=== Project Management 

*[시스템 관리] > [Project Management]* 메뉴를 선택하면 사용자가 생성한 프로젝트 목록을 조회하고 관리할 수 있다.

프로젝트 관리 화면에서 [+프로젝트] 버튼을 클릭하면 신규 프로젝트를 생성할 수 있다. 프로젝트 목록의 '실행' 항목에서 image:resources/icon_edit.png[width=20] 아이콘을 클릭하면 프로젝트를 수정할 수 있다. 프로젝트 목록에서 프로젝트를 선택한 후 [삭제] 버튼을 클릭하거나 '실행' 항목에서 image:resources/icon_delete.png[width=20] 
아이콘을 클릭하면 프로젝트를 삭제할 수 있다.


목록 위에 '검색' 입력 항목에 검색할 프로젝트 이름을 입력하고 <Enter> 키를 누르거나 image:resources/icon_search.png[width=22] 아이콘을 클릭하면 해당 프로젝트를 조회할 수 있다. 목록 아래에 조회된 건수와 페이징 정보, 조회할 건수를 선택할 수 있다.

목록에서 '프로젝트명'을 클릭하면 <<figure_project_detail, 프로젝트 정보 화면>>에서 해당 프로젝트의 상세 정보를 조회할 수 있다. 자세한 내용은 <<chapter_project,"제3장 프로젝트">>를 참고한다.

[#sect_project_create]
==== 프로젝트 생성

<<figure_project_list, 프로젝트 관리 화면>>의 [+프로젝트] 버튼을 클릭하거나 초기 <<figure_main, 메인 화면>>에서 [새 프로젝트 생성] 버튼을 클릭하면 프로젝트를 생성할 수 있다. 각 항목을 입력하고 [적용] 버튼을 클릭한다.
 
image::resources/figure_project_create.png[title="[Project Management] - 프로젝트 생성", width=450]

다음은 입력 항목에 대한 설명이다.

[width="100%",cols="3,7",options="header"]
|==========
|항목      	|설명
|프로젝트명	|프로젝트의 이름이다. '프로젝트명' 항목은 중복된 값을 입력하면 에러가 발생하므로 주의한다.
|설명       |프로젝트의 목적, 용도 등의 설명을 입력한다.
|==========

[#sect_admin_proj_edit]
==== 프로젝트 수정

<<figure_project_list,프로젝트 관리 화면 >>의 프로젝트 목록에서 '실행' 항목의 image:resources/icon_edit.png[width=20] 아이콘을 클릭하면 프로젝트 정보를 수정할 수 있다. [참여자 수정] 버튼을 클릭하면 프로젝트에 참여 중인 사용자에 대한 정보를 조회하거나 추가, 삭제할 수 있다(<<figure_project_edit_member, 프로젝트 참여자 수정>>  참고). 각 항목을 입력한 후 [적용] 버튼을 클릭한다.

 
image::resources/figure_project_edit.png[title="프로젝트 수정", width=600]

참여자 수정 화면에서 멤버 정보를 변경한 후 [확인] 버튼을 클릭하면 프로젝트에 선택한 참여자의 정보가 입력된다.

[#figure_project_edit_member] 
image::resources/figure_project_edit_member.png[title="프로젝트 참여자 수정", width=600]

[#sect_admin_proj_delete]
==== 프로젝트 삭제

<<figure_project_list, 프로젝트 관리 화면>>의 목록에서 프로젝트를 선택한 후 [삭제] 버튼을 클릭하거나 '실행' 항목의 image:resources/icon_delete.png[width=20] 아이콘을 클릭하면 프로젝트 정보를 삭제할 수 있다. 목록에서 삭제할 정보를 확인한 후 [삭제] 버튼을 클릭한다.
 
image::resources/figure_project_delete.png[title="프로젝트 삭제", width=300]

 
NOTE: 삭제하려는 프로젝트에 타 프로젝트와 공유된 데이터 오브젝트 또는 #Default DataSource와 #SQL 이외의 데이터소스, 데이터 객체 및 Data Job, Service Job이 생성되어 있는 경우에는 삭제되지 않는다. 프로젝트를 삭제하려면 해당 프로젝트의 데이터소스와 Job을 먼저 삭제해서 빈 프로젝트(Empty Project) 상태로 만들어야 한다.
